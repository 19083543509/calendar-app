# 本地日历记事本

一个桌面日历软件，整页显示月份，每个日期格子可以记事，记事以小字显示在日历上。

## 技术栈

- **语言**: Python 3 (tkinter)
- **GUI**: tkinter（标准库，零额外依赖）
- **数据存储**: JSON 文件（`notes.json`）
- **日历逻辑**: `calendar` 标准库

## 项目结构

```
D:/cc/calendar_app/
├── main.py              # 入口，启动 tkinter 主循环
├── calendar_app.py      # 日历 UI 核心（导航 + 网格 + 编辑弹窗）
├── storage.py           # JSON 数据读写封装
└── notes.json           # 自动生成，记事持久化文件
```

## 启动方式

```bash
cd D:/cc/calendar_app && python main.py
```

## 功能说明

- 月视图日历网格（周一~周日）
- ◀ ▶ 按钮切换上/下月
- 「今天」按钮快速回到当前月
- 当天日期蓝底高亮
- 点击任意日期 → 弹出编辑窗口，每行一条记事
- 保存后记事以小字显示在对应格子（最多 3 条，超出显示 +N）
- 数据自动保存到 `notes.json`，重启不丢失

## 数据格式

```json
{
  "2026-05-10": ["下午3点开会", "买牛奶"],
  "2026-05-15": ["交报告"]
}
```
